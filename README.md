# Default Vaadin-Project with usage of Meecrowave

## How to use ##

#### Run in IDE/Eclipse ####

1. `clean install`

2. `Right click MainStarter.java → Run as → Java Application`

#### Run with Maven ####

1. `clean install meecrowave:run`
